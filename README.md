## Installation

```bash
$ npm install
```

## Build 
```bash
$ npm run build
```

## Usage

```bash
$ npm run dev
```

## Environment variables
- `SECRET_KEY` - secret key for JWT
- `PORT` - port to run the server on
- `DB_HOST` - database host
- `DB_USER` - database user
- `DB_PASSWORD` - database password
- `DB_NAME` - database name
